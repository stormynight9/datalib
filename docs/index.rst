Welcome to DataLib's documentation!
================================

**DataLib** is a Python library designed to simplify data manipulation and analysis. It provides features for data processing, statistical analysis, visualization, and machine learning.

Installation
-----------

You can install DataLib using pip:

.. code-block:: bash

   pip install datalib
